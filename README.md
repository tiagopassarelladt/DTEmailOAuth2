# DTEmailOAuth2 - Envio de Emails com Autenticação em Dois Fatores 🔒📧

O **DTEmailOAuth2** é um componente desenvolvido para facilitar o envio de emails com suporte à autenticação em dois fatores (OAuth2). Ele utiliza a poderosa biblioteca **Synapse** e mantém total compatibilidade com as propriedades do **ACBrMail**, permitindo uma transição suave para quem já utiliza o ACBr para envio de emails. 

Com isso, você pode implementar segurança adicional na comunicação por email sem alterações drásticas no seu código, mantendo a familiaridade e confiabilidade do ACBrMail.

---

### 🔑 **Por que usar o DTEmailOAuth2?**
- **Compatibilidade Total com ACBrMail**  
  Mantenha o mesmo padrão de código e propriedades, sem necessidade de aprendizado adicional.  
- **Autenticação via OAuth2**  
  Atenda às exigências modernas de segurança com autenticação em dois fatores, essencial para serviços como Gmail, Outlook e muitos outros.  
- **Baseado na biblioteca Synapse**  
  Aproveite a eficiência e simplicidade da biblioteca Synapse, reconhecida pela comunidade Delphi.

---

### 🚀 **Facilidade de Migração**  
Para quem já utiliza o **ACBrMail**, a migração para o **DTEmailOAuth2** é praticamente automática, pois o componente foi projetado para oferecer as mesmas propriedades e comportamentos.
